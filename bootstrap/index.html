HTML

Hyper Text Markup Language

Standard "språket" för alla webbläsare.

ID

ID är en attribut inom HTML5, och används för att specifiera ett unikt id för en element. Den skiljer sig från

Den är liksom "class" attributen, men är döpt på ett annat sätt.

Inom CSS så skrivs ID med en "#".

Class

Class är en attribut inom HTML5, och används för att specifiera en eller flera klasser inom en element.

Den är liksom "ID" attributen, men är döpt på ett annat sätt.

Inom CSS så skrivs Class med en ".".

Meta

Meta är en tag som definierar metadata inom en HTML fil. Metadata är information om data.

Meta taggen går alltid in i head taggen, vilket kommer inte att visas på webbsidan. Datan används av webbläsaren.

Exempel på metadata som används:

  1. charset
  2. content
  3. name

Responsive

Responsive är en beskrivning av en webbsida som mottager förändringar på skärmen, d.v.s. om man zoomar in eller zoomar ut så borde webbsida mottaga informationen.

Responsive webbsidor oftas använder en speciell kod i "head" taggen, vilket man använder sig av meta taggen.


HTTP

HTTP står för "Hyper Text Transfer Protocol" och används för att klienter och sevrar ska kunna kommunicera med varandra på the world wide web.


HTTPS

HTTPS, liksom HTTP, betyder "Hyper Text Transfer Protocol Secure". Den är typ samma som HTTP, fast den är krypterad och säkrare.


Compressed Data

Compressed data är data som omkodas på ett sätt som gör att färre informationsbärande enheter behöver användas. Oftas tillämpar man detta inom webbservrar.


CDN

CDN, som står för "Content Delivery Network", utses av geografiskt utspridda nätverk av proxy-servrar och deras datorhallar. vsikten med dessa nätverk är att tillhandahålla nättjänster nära användare, för ökad driftsäkerhet och prestanda samt minskad latens.


Local Bootstrap

Local bootsrap i principt en bootstrap bibliotek installerad lokalt på datorn. Just nu så används denna webbsida sig av bootsrap biblioteket!

JQuery

JQuery är en biblitek skriven med javascript, och används inom webbutveckling för att gör det lättare för att tillämpa javascript kod inom HTML elementer. Knappen nedanför demonstrerar JQuery applikationen
